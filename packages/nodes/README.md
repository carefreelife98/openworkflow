# @openpipeline/nodes

Execution kernel (compiler, node-runner, registry, binding resolver) and built-in IF/LLM nodes for OpenPipeline.

Part of [OpenPipeline](https://github.com/carefreelife98/openworkflow) — a framework-agnostic engine for compiling and running MCP-tool pipelines as LangGraph DAGs.

## Install

```bash
npm i @openpipeline/nodes
```

## Usage

The execution kernel: the DAG→LangGraph compiler, node runner, registry, value-binding resolver, and the built-in `IF` / `LLM` nodes. Author custom nodes with `defineNode` from `@openpipeline/core`.

See the [root README](https://github.com/carefreelife98/openworkflow#readme) for the full quickstart and the playground.
