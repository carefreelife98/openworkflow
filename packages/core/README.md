# @openpipeline/core

Framework-agnostic execution kernel and interface contracts for OpenPipeline.

Part of [OpenPipeline](https://github.com/carefreelife98/openworkflow) — a framework-agnostic engine for compiling and running MCP-tool pipelines as LangGraph DAGs.

## Install

```bash
npm i @openpipeline/core
```

## Usage

Provides the `NodeSpec` contract, `PipelineStore`/`StepRecorder`/`LlmFactory`/`CatalogLoader` interfaces, the run-state types, and pure helpers. You rarely install this alone — it comes in via the other packages.

See the [root README](https://github.com/carefreelife98/openworkflow#readme) for the full quickstart and the playground.
